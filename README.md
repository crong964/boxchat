# ChatBox


### Description

- ChatBox is  system help system to send messages, image, location quickly and effectively, i use  ReactJS, TailwindCSS for frontend and use NodeJs (ExpressJs) for backend, firebase to store image
- The database I choose for this project is MySQL.
